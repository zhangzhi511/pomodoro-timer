# 🍅 番茄钟 Pomodoro Timer

一个简洁的桌面番茄钟应用，基于 Python + tkinter 实现，支持深色主题与圆形进度动画。

## 功能特性

- **三种模式**：专注（25 min）、短休息（5 min）、长休息（15 min）
- **圆形进度条**：动态显示剩余时间比例
- **自动切换**：每完成 4 个番茄自动进入长休息
- **提示音**：阶段结束时发出蜂鸣提醒
- **弹窗通知**：切换阶段时弹出提示
- **番茄计数**：记录今日完成数量
- **深色主题**：护眼配色，长时间使用不疲劳

## 环境要求

- Windows 系统
- Python 3.8+
- 无需安装第三方库（tkinter、winsound 均为标准库）

## 快速开始

```bash
python pomodoro_timer.py
```

## 使用说明

| 操作 | 说明 |
|------|------|
| 点击 **▶ 开始** | 启动倒计时 |
| 点击 **⏸ 暂停** | 暂停计时 |
| 点击 **↺ 重置** | 重置当前阶段 |
| 顶部标签切换 | 手动切换专注 / 短休息 / 长休息 |

## 番茄工作法说明

番茄工作法由 Francesco Cirillo 于 1980 年代提出：

1. 选择一项任务
2. 专注工作 **25 分钟**（一个番茄）
3. 休息 **5 分钟**
4. 每完成 **4 个番茄**，休息 **15 分钟**
